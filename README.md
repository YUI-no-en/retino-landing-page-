# RETINO - スキンケア商品ランディングページ

## 概要
レチノール美容液「RETINO」のランディングページです。
モダンなデザインとユーザビリティを重視した設計になっています。

## 技術スタック
- HTML5
- CSS3
- レスポンシブデザイン
- CSS Grid & Flexbox

## 特徴
- ✨ モダンなグラデーションデザイン
- 📱 モバイルファースト・レスポンシブ対応
- 🎨 美しいアニメーション効果
- 🚀 高速な読み込み速度

2. ブラウザでindex.htmlを開く
